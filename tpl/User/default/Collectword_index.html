中奖算法如下：
所有字集齐为中奖，抽中最后一个字的概率即为中奖概率。
中奖概率 = （1 / 收集文字个数（4-8））*（每天发放奖品数 / 预计参与人数）
注：1.每日发放的奖品数 = 剩余奖品数量 / 活动剩余天数；
　　2.每天发放奖品数大于预计人数时为中奖概率率为1，因为1人只能领取一个奖品。
